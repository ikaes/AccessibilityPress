== Description ==

Allow People with disabilities read and navigate your wordpress site. 

Make your wordpress site accessible to a larger audience group with this most advanced accessibility plugin. Accessibility Press works instantly as you install and enable it, without any complicated configuration.

This helps your site with AA rating of WCAG 2.0.

**AccessibilityPress works on any wordpress site built on any theme or language**

[youtube https://www.youtube.com/watch?v=dTHQkSfb0g8]

<strong>Accessibility Features:</strong>

Adds a toolbar with following features to your site:

* Resize font (increase/decrease)
* Big Light/Dark pointer
* Content Magnification
* Link & headline highlight
* Image tooltip
* Greyscale
* Negative Contrast
* Content navigation with keyboard
* Block animations

<strong>Customizable Options:</strong>

From accessibility menu on the dashboard you can set options like:

* Left/right positioning of Accessibility toolbar
* Toolbar on mobile devices
* Language options RTL/LTR
* Set all button title in your language
* Custom toolbar color choice to match your theme colors

== Installation ==

Installing this plugins is just like installing any other WordPress plugins.

== Frequently Asked Questions ==

= Is the plugin compatible for any language? =

Yes, The plugin can be used on any language(RTL/LTR) sites and all the label can be defined in any language.

= How do i set it up for a custom language? =

Easy, go to the dashboard > accessibility then choose language "Other/Custom", select language direction and enter all the labels in your language then finally save options.

= Is this plugin going to make my site AA level of W3CAG? =

This plugin will help you improve the score and it provides many accessibility control options but this does not guarantee AA level of W3CAG
